# PhotoScanner Web

A browser-based photo and document scanner with 4-point crop, live preview, filters, and high-quality export.

Live demo: [https://eeness23.github.io/photoscanner-web/](https://eeness23.github.io/photoscanner-web/)

This project was created through AI-assisted prompt-based development.

## Features

- 4-point perspective crop
- Zoom loupe for precise corner placement
- Live preview
- Filter presets and manual adjustments
- PNG, JPEG, and PDF export
- Multilingual UI
- Light and dark mode

## Tech

- HTML
- CSS
- JavaScript
- Canvas API
- jsPDF

## Run

Open the HTML file in your browser.
